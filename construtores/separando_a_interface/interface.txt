uma das vantagens de se empacotar corretamente classes é que se pode reutilizá-las, um exemplo é a classe string

todo programa só pode ter um main, entao, se toda classe tivesse um main isso impediria que reutilizássemos essa classe, por isso separamos os arquivos.
